Hero Graphics
Usage Rules
  1. All Hero Graphics should incorporate each of these three elements: Data + Time + Action
  2. The following options for each element are available to use in your compositions

Data

Data can be conceptual in the form of dimensional diagrams, or more literal representations of the various industries we serve and the types of data they generate



Time

A layer of time must be overlaid on top of our data layer; this can come in the form of timelines, stopwatch dials, radars, or equalizers



Action

The action layer pinpoints a specific moment in time where the insight is derived; the moment must represent ‘now’, and data beyond that point should only be a suggestion (ie: a prediction)

Nuetrals

There are many nuetrals because they are the most used. Having subtle differences between the dark and light end of the spectrum is extremely useful when building UI.

The nuetrals follow two main principles: never use true black, and never use true nuetrals. Our nuetral palette is subtley colored with brand colors, the dark end of the spectrum is colored with Comet, and the light end is colored with Pool.
